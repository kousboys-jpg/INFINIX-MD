## ⚙️ Fonctionnalités

- **Tag tous les membres** avec la commande `.tagall`
- **Commandes réservées aux admins** (certaines commandes ne marchent que pour les admins)
- **Jeux** comme Tic-Tac-Toe pour animer le groupe
- **Text-to-Speech** avec `.tts`
- **Création de stickers** avec `.sticker`
- **Détection anti-lien** pour sécuriser le groupe
- **Warn et gestion des membres** avec contrôle admin

---

## 📖 À propos

Le bot **Knight WhatsApp Bot** aide les admins de groupes en leur fournissant des outils pour gérer efficacement de grands groupes WhatsApp.  
Il utilise la librairie **Baileys** pour interagir avec WhatsApp Web et supporte le mode multi-appareil.

Il est léger et facilement personnalisable : tu peux ajouter des commandes selon tes besoins.  
Le bot tourne sous **Node.js** et propose l’authentification via QR Code pour lier ton compte WhatsApp.

---

## 🛠️ Installation & configuration

### Prérequis

- Node.js installé
- Git installé (pour cloner le dépôt)

### Étapes d’installation

1. **Cloner le dépôt :**

    ```bash
    https://github.com/RebelleMasque/INFINIX-MD.git
    cd INFINIX-MD
    ```

2. **Installer les dépendances :**

    ```bash
    npm install
    ```

3. **Lancer le bot :**

    ```bash
    node index.js
    ```

4. **Scanner le QR code :**

    Quand le bot démarre, un QR code s’affiche dans le terminal.  
    Scanne-le depuis WhatsApp : **Appareils connectés** pour connecter ton compte.
